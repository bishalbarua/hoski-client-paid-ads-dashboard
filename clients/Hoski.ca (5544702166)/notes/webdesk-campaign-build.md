## Keywords NOT Included (and Why)

| Keyword | Intent | Reason Excluded |
|---------|--------|-----------------|
| how to build a website | Informational/DIY | Zero purchase intent — added as negative |
| web design software | Commercial — wrong category | They want software, not an agency |
| web designer freelancer | Transactional | Different buyer — looking for freelance, not agency. Could cannibalise but lower ATV. Add to competitor campaign later. |
| web design pricing | Commercial investigation | Price-first intent signals low-quality lead for a premium agency. Test later if volume needed. |
| web design course | Educational | Not buying a service — add as negative |
| seo agency | Adjacent service | Hoski's Web Desk doesn't sell SEO. Negated to avoid irrelevant leads. |
| social media agency | Adjacent service | Not offered on Web Desk. Negated. |
| google ads agency | Adjacent service | Different Hoski service (Performance). Separate campaign if needed. |
| website builder | DIY intent | They want a self-service tool, not an agency. |

## 30-Day Post-Launch Review Checklist

- [ ] Run search term reports weekly — add irrelevant queries as phrase negatives
- [ ] Check impression share by ad group — if under 50%, either raise budget or pause lowest-performing group
- [ ] Pause RSA headlines marked "poor" after 2,000+ impressions — replace with new variants
- [ ] Identify any high-volume search terms to promote as exact match keywords
- [ ] Flag any ad groups with CTR under 3% — test new RSA copy
- [ ] Review geo performance — if Montreal/Toronto/Vancouver outperform, increase bid adjustment to +25%
- [ ] Check device performance — if mobile CTR is high but conversion low, add landing page speed fix to roadmap
- [ ] After 30+ conversions per campaign — switch bid strategy to Maximize Conversions

## Pre-Launch Checklist

- [ ] Conversion tracking live and confirmed firing (strategy call booked event)
- [ ] CallRail phone number added to call assets
- [ ] Trustpilot score confirmed — update all H9 headlines with actual rating
- [ ] Negative keyword lists uploaded at account level (universal) and campaign level
- [ ] Cross-negatives added at ad group level
- [ ] All RSAs have minimum 2 ads per ad group (build 2 RSA variants per ad group = 12 total RSAs across both campaigns)
- [ ] Sitelinks live with descriptions on all 4 minimum sitelinks per campaign
- [ ] Callout extensions added (8 minimum per campaign)
- [ ] Structured snippets added
- [ ] Display Network checkbox confirmed OFF
- [ ] Search Partners: start ON, review after 30 days (can exclude if CTR/CVR diverges)
- [ ] Ad rotation: Optimize (not "do not optimize")
- [ ] URL tracking parameters added to all final URLs: `?utm_source=google&utm_medium=cpc&utm_campaign={campaignid}&utm_content={adgroupid}&utm_term={keyword}`
