Google Ads Channel
Google Ads Strategist
/google-strategist
NEW

Purpose

Designs the plan. Owns all strategic decisions for Google: campaign architecture, keyword strategy, bid strategy, audience approach. Produces a brief that the Google Manager executes.

Inputs
  • client-info.md
  • Budget and CPA targets
  • Business context / URL
  • Historical performance data
Outputs
  • Campaign structure brief
  • Keyword strategy + intent map
  • Bid strategy recommendation
  • Audience segmentation plan
Wraps these existing agents
google-campaign-architect google-keyword-intelligence google-bid-budget-optimizer google-audience-architect google-pmax-intelligence
Google Ads Manager
/google-manager
NEW

Purpose

Executes and monitors. Implements what the Strategist designed. Runs all weekly operational tasks: pacing, search terms, copy performance, negatives, bid health. Reports anomalies upward.

Inputs
  • Strategy brief from Strategist
  • Live account data / API
  • Search term exports
  • Weekly performance snapshots
Outputs
  • Weekly action list
  • Search term report (4 sections)
  • Pacing alert if off-track
  • Copy swap recommendations
Wraps these existing agents
google-account-health-monitor google-search-terms-analyst google-budget-pacing google-negative-keyword google-ad-copy-strategist google-reporting-analyst

Consolidates: /weekly-check + /search-terms + /ad-copy-testing-analyzer into one role

Meta Ads Channel
Meta Ads Strategist
/meta-strategist
NEW

Purpose

Designs the Meta plan. Owns campaign architecture, full-funnel structure (ToFu/MoFu/BoFu), audience strategy, and creative direction. Produces briefs for Meta Manager and Creative Strategist.

Inputs
  • client-info.md
  • Funnel goals and budget
  • Pixel / conversion event status
  • Existing audience data
Outputs
  • Campaign architecture
  • Audience strategy (cold/warm)
  • Creative brief for Strategist
  • Bid and budget allocation
Wraps these existing agents
meta-campaign-strategist meta-audience-architect meta-creative-strategist meta-bid-budget-optimizer meta-conversion-optimizer
Meta Ads Manager
/meta-manager
NEW

Purpose

Executes and monitors Meta campaigns. Tracks creative fatigue, pacing, pixel health, audience performance, and scaling signals. Flags when creative needs refresh or audiences are exhausted.

Inputs
  • Strategy brief from Strategist
  • Live Meta account data
  • Creative performance signals
  • Pixel / event data
Outputs
  • Weekly action list
  • Creative fatigue alerts
  • Scaling signals (up/down)
  • Pixel health report
Wraps these existing agents
meta-account-health-monitor meta-creative-performance-analyst meta-scaling-diagnosis meta-pixel-events-guardian meta-ad-library-intelligence
Cross-Channel Specialists
Creative Strategist
/creative-strategist
NEW

Purpose

Full creative ownership across both channels. Not just copywriting — owns the entire creative layer: audit what's running, analyze what's working, ideate new angles, brief concepts for image/video/copy, build the testing framework.

Audit

  • RSA asset performance (BEST/GOOD/LOW)
  • Meta creative fatigue signals
  • Winning pattern extraction

Ideation

  • New copy angles and hooks
  • Image and video concepts
  • UGC direction

Strategy

  • Creative testing framework
  • Awareness continuum mapping
  • Competitive creative gap analysis

CRO Strategist
/cro-strategist
NEW

Purpose

Full landing page ownership. Audits current pages, identifies conversion leaks, ideates improvements, builds the CRO strategy. Ensures message match between ads and landing pages across all channels.

Audit

  • Above-the-fold analysis
  • Message match vs. ad copy
  • Trust signal inventory
  • Mobile experience check

Ideation

  • Headline and CTA alternatives
  • Layout recommendations
  • Social proof placement
  • Form friction reduction

Strategy

  • Per-campaign LP prescriptions
  • A/B test prioritization
  • Funnel drop-off diagnosis
  • Post-click experience map
Wraps these existing agents
cross-landing-page-cro
