# PPC Domain Knowledge Reference

Quick reference for common PPC concepts when building skills.

## Universal Waste Patterns (Safe to Negative)

These patterns indicate non-commercial intent across virtually all industries:

**Employment Intent**
- jobs, careers, hiring, salary, glassdoor, indeed, employment, work for, recruit

**DIY/Educational Intent**
- how to, tutorial, DIY, course, training, certification, learn, template, example

**Definitional Intent**
- what is, definition, meaning, wiki, wikipedia

**Free/Cheap Seekers (context-dependent)**
- free, cheap, discount, coupon (evaluate based on business model)

## Protected Patterns (Never Auto-Negative)

**Commercial Investigation**
- best [service], top [service], [service] reviews, [service] near me, compare [service]
- These indicate active buying consideration

**Comparison Shopping**
- [service] vs [competitor], [service] or [alternative], which [service]
- User is deciding between options—high intent

**Brand + Service Combinations**
- Any term containing both brand name AND service offered
- Even if other "waste" signals present

## Match Type Decision Framework

| Scenario | Match Type | Level |
|----------|------------|-------|
| Universal waste (jobs, DIY) | Broad | Account |
| Industry-specific waste | Phrase | Campaign |
| Competitor names | Phrase | Campaign |
| Ambiguous terms | Exact | Ad Group |
| Geographic mismatch | Phrase | Campaign |

## Common Metrics & Benchmarks

**Search Campaign Benchmarks (vary by industry)**
- CTR: 2-5% typical, <1% concerning, >7% excellent
- CVR: 2-5% typical for lead gen, 1-3% for ecommerce
- Quality Score: 7+ good, 5-6 acceptable, <5 needs work

**Statistical Significance Thresholds**
- Minimum conversions for confidence: 30+
- Minimum clicks for CTR analysis: 100+
- Minimum impressions for visibility: 1,000+

**Data Freshness**
- Search terms: Last 30 days typical
- Performance trends: 7-day, 30-day, 90-day comparisons
- Seasonality: Year-over-year when available

## Intent Classification

**Bottom Funnel (High Intent)**
- Price/cost queries, "near me", brand + service, "buy", "hire", "get quote"
- Action: Aggressive bidding, direct CTAs

**Mid Funnel (Consideration)**
- Comparison queries, "best", "top", "reviews", "vs"
- Action: Educational content, trust signals

**Top Funnel (Awareness)**
- "What is", "how to", informational queries
- Action: Usually exclude from Search, consider for Display/YouTube

## Common PPC Data Formats

**Google Ads Exports**
- Search Terms Report: term, impressions, clicks, cost, conversions
- Keyword Report: keyword, match type, QS, status, metrics
- Campaign Report: campaign, budget, status, metrics by date

**Typical Column Names (handle variations)**
- Spend/Cost/Amount Spent
- Conversions/Conv./Leads
- CPA/Cost per Conversion/Cost/Conv.
- ROAS/Return on Ad Spend/Conv. Value/Cost

## RSA Headline Allocation

Recommended distribution for 15 headlines:
- 2-3 Keyword headlines (target keyword variations)
- 3-4 Benefit headlines (outcomes, not features)
- 2-3 Social proof headlines (numbers, ratings, trust)
- 2-3 CTA headlines (action-focused)
- 2-3 Differentiator headlines (unique value)
- 1-2 Urgency/scarcity (if applicable)

Character limits:
- Headlines: 30 characters
- Descriptions: 90 characters
- Display path: 15 characters each

## Landing Page CRO Priorities

**Above the Fold (Critical)**
1. Headline clarity (matches ad promise)
2. Primary CTA visibility
3. Trust signals (logos, ratings, guarantees)

**Message Match Scoring**
- Exact keyword in headline: +2
- Benefit alignment: +2
- CTA consistency: +1
- Visual consistency: +1
- Score 5-6: Excellent, 3-4: Acceptable, <3: Needs work

**Mobile Considerations**
- CTA visibility without scroll
- Phone number clickable
- Form fields minimized
- Load time <3 seconds
