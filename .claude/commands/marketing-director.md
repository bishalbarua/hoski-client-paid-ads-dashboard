# Marketing Director

Orchestrate the full marketing specialist team on any paid media task.

## Usage

`/marketing-director [task or question]`

## What It Does

The Marketing Director receives your request, applies the Dependency Graph and Scope Filter to determine which specialists are needed, issues each one a scoped brief, synthesizes all outputs into a unified action plan, runs an adversarial QA review, and delivers a single coherent final report.

**The full team:**
- Keyword Intelligence Agent (proactive keyword architecture)
- Search Terms Analyst (reactive query mining from existing data)
- Campaign Architect (campaign structure, segmentation, budget logic)
- Ad Copy Strategist (RSA headlines, angles, pin strategy)
- Bid & Budget Optimizer (smart bidding, tCPA/tROAS math)
- Competitive Intelligence Agent (SERP signals, competitor analysis)
- QA Specialist (adversarial review of all outputs before delivery)
- Meta specialists (when Meta scope is included)

Not all specialists run every time. The Director scopes the team to match the request.

## When to Use

- Full account builds or campaign builds from scratch
- Strategic audits across multiple domains
- Performance diagnosis that spans bidding, structure, and copy
- "What should we prioritize this month?" questions
- Any complex request where a single specialist is not enough

## When NOT to Use

| Use this instead | When |
|---|---|
| `/weekly-check` | Weekly operational review |
| `/ppc-account-health-check` | One-time strategic health snapshot |
| `/search-terms` | Dedicated search terms sweep |
| `/monthly-report` | Monthly client report |
| `/ads-strategy-architect` | Strategy from a new business URL |

## Examples

```
/marketing-director audit the dental account and tell me what to prioritize this month

/marketing-director build a Google Ads search campaign for a new personal injury law client

/marketing-director why is CPA rising on the Park Road account and what do we do about it

/marketing-director plan the first 90 days for FaBesthetics on Google and Meta

/marketing-director the Desai Law campaigns are not scaling — run the full team and tell me why
```

## What You Get

A single final report with:
- Which specialists ran and what they found
- The strategic picture across all domains
- Any cross-specialist conflicts and how they were resolved
- Priority 1 / 2 / 3 action plan
- Decisions that need your input
- What not to touch
- QA verdict (PASS / CONDITIONAL PASS / FAIL)
