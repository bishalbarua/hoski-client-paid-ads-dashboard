Never use em dashes (--, —) in any output, including responses, reports, documents, or code comments.

**Why:** User preference for cleaner writing style.

**How to apply:** Replace em dashes with commas, colons, parentheses, or restructure the sentence. Applies to all outputs across all contexts.
