---
name: Always Use Exact Campaign Names
description: When summarizing Google Ads data, always reference the exact campaign name from the data, never a generic label
type: feedback
---

Always use the exact campaign name when summarizing results (e.g. "Hoski | Search | Dental Patient Acquisition 14/3/26"), never a generic substitute like "main campaign", "the search campaign", or "active campaign".

**Why:** Generic names are ambiguous and make it harder to act on the finding without cross-referencing the UI.

**How to apply:** Any time you summarize API output, copy the campaign name exactly as it appears in the data.
